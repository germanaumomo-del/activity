歡迎使用出席統計網頁！

由於您正在外部平台（如 GitHub Pages）執行本程式，網頁需要連接到您專屬的 **Firebase Firestore** 即時資料庫，以便進行多裝置即時同步與儲存。

💡 5分鐘快速免費建立資料庫：

  1. 前往 Firebase 主控台 建立一個專案。
  2. 在專案內新增「網頁應用程式 (Web App)」。
  3. 複製網頁顯示的 firebaseConfig 設定 JSON 碼。
  4. 同時於「Firestore Database」中點擊「建立資料庫」，並設定為測試模式 (Test Mode)。
這會作為您在資料庫中的集合根路徑，避免與他人資料混雜。
